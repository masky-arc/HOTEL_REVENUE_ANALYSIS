Hotel Revenue Analysis Dashboard

 Project Overview

The **Hotel Revenue Analysis Dashboard** is a Power BI project designed to analyze hospitality performance metrics such as revenue, occupancy, bookings, and customer trends. The dashboard provides interactive visualizations that help stakeholders understand hotel performance, identify revenue opportunities, and support data-driven decision-making.

This project focuses on transforming raw hotel booking data into meaningful insights using Power BI visualizations and data modeling.

Objectives

* Analyze hotel revenue performance.
* Monitor key hospitality metrics such as occupancy rate and booking trends.
* Identify high-performing properties and time periods.
* Support strategic decisions for pricing, marketing, and operations.

## Dashboard Features

The Power BI dashboard includes:

### Revenue Insights

* Total revenue generated
* Revenue trends over time
* Revenue comparison by hotel or property

### Booking Performance

* Total bookings
* Booking trends by day, week, or month
* Cancellation analysis

### Occupancy Analysis

* Occupancy rate by property
* Room utilization patterns
* Seasonal demand analysis

### Customer & Market Insights

* Customer booking patterns
* Demand across different hotel locations
* Performance comparison between properties
  

## Key KPIs

The dashboard tracks important hospitality metrics including:

* Total Revenue
* Total Bookings
* Occupancy Rate
* Average Daily Rate (ADR)
* Revenue per Available Room (RevPAR)
* Cancellation Rate


## Tools & Technologies

 **Power BI Desktop** – Data modeling and dashboard creation
 **Power Query** – Data transformation and cleaning
 **DAX (Data Analysis Expressions)** – KPI calculations and measures
 **Data Visualization** – Interactive charts, filters, and dashboards


## Data Model

The dataset typically includes multiple tables such as:

* Hotels / Properties
* Bookings
* Rooms
* Date / Time
* Revenue and pricing data

Relationships between these tables enable detailed performance analysis.


## Insights Generated

This analysis helps answer key business questions:

* Which hotels generate the highest revenue?
* What time periods have the highest booking demand?
* How do cancellations affect revenue?
* What are the occupancy trends across properties?


## How to Use the Dashboard

1. Open the **`HOTEL_revenue_analysis.pbix`** file in **Power BI Desktop**.
2. Refresh the dataset if needed.
3. Navigate through the report pages.
4. Use filters and slicers to explore specific time periods or properties.
